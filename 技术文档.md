## 一、项目概述

实时多人德州扑克游戏，支持 2–10 人同房对战，具备完整的下注逻辑（翻牌前→翻牌→转牌→河牌→摊牌）、边池计算、断线重连、观战/准备系统和嘲讽互动等功能。

## 六、游戏流程

```
waiting（等待玩家加入）
    │  房主点击开始游戏
    ▼
preflop（翻牌前）
  · 发两张底牌给每位玩家
  · 小盲注 & 大盲注自动下注
  · 从大盲注左侧开始行动
    │  所有玩家行动完毕
    ▼
flop（翻牌）
  · 翻出3张公共牌
  · 从小盲注开始行动
    │
    ▼
turn（转牌）
  · 再翻1张公共牌
    │
    ▼
river（河牌）
  · 再翻1张公共牌
    │  下注结束 或 剩余玩家全部 all-in
    ▼
showdown（摊牌）
  · 用 pokersolver 评估每位玩家最佳5张
  · 边池分配（calculateSidePots）
  · 向所有玩家广播结果（胜者、盈亏、牌型）
    │
    ▼
settlement（结算，60秒超时）
  · 每位玩家选择：准备下一局 / 观战
  · 超时自动标记为准备
    │  所有非观战玩家都点了准备
    ▼
waiting（下一局）
  · 庄家位轮转
  · 观战玩家不参与发牌

特殊路径：
  preflop/flop/turn/river → 仅剩1名玩家未弃牌 → 直接获胜，不摊牌
  任意街 → 所有活跃玩家均 all-in → 自动翻完剩余公共牌 → showdown
```

## 八、下注系统实现

### 行动类型
```javascript
switch (action) {
  case 'fold':   // 弃牌，player.folded = true
  case 'check':  // 过牌（当前无人下注时有效）
  case 'call':   // 跟注（筹码不足则自动 all-in）
  case 'raise':  // 加注（重置其他玩家 hasActed = false）
  case 'allin':  // 全下
}
```

### 轮次推进条件
满足以下两个条件后自动推进到下一街：
1. 所有未弃牌玩家的 `hasActed === true`
2. 所有未弃牌玩家的 `bet` 相等（或为 all-in 状态）

### 边池计算（calculateSidePots）
```
示例：玩家A投入50，玩家B投入100，玩家C投入200
  主池 = 50 × 3 = 150（A/B/C都有资格赢）
  副池1 = (100-50) × 2 = 100（B/C有资格赢）
  副池2 = (200-100) × 1 = 100（仅C有资格赢）
```

## 十、断线重连机制

```javascript
// server/game/roomManager.js
joinRoom(socketId, roomId, nickname) {
  const room = rooms.get(roomId);
  // 同昵称玩家 → 视为重连
  const existing = room.players.find(p => p.nickname === nickname);
  if (existing) {
    existing.socketId = socketId;       // 更新 socket ID
    existing.disconnected = false;      // 清除断线标记
    return { reconnected: true, room };
  }
  // 否则新加入玩家（需在 waiting 阶段）
}
```

断线玩家：
- 游戏进行中自动弃牌（超时后 `timedOut` 触发）
- 头像置灰，显示"断线"标记
- 相同昵称重新加入后立刻恢复状态

## 十一、手牌评估

前后端均使用 `pokersolver` 库：

```javascript
const { Hand } = require('pokersolver');

// 从7张牌中自动选最优5张
const hand = Hand.solve([...holeCards, ...communityCards]);
hand.name    // "Two Pair"
hand.descr   // "Two Pair, Aces and Kings"
hand.rank    // 数字，越大越强（用于排序）

// 比较多位玩家，返回胜者数组
const winners = Hand.winners([handA, handB, handC]);
```

**牌型排名（高→低）：**
1. Royal Flush（皇家同花顺）
2. Straight Flush（同花顺）
3. Four of a Kind（四条）
4. Full House（葫芦）
5. Flush（同花）
6. Straight（顺子）
7. Three of a Kind（三条）
8. Two Pair（两对）
9. Pair（一对）
10. High Card（高牌）

## 十二、倒计时实现

```javascript
// server/timerManager.js
{
  timeout: NodeJS.Timeout,     // setTimeout 句柄
  startTime: Date.now(),       // 开始时间戳
  duration: 20,                // 总时长（秒）
  roomId: "ABC123",
```

- 客户端收到 `timerStarted` 后用 `startTime + duration` 本地计算剩余时间

## 十五、核心特性总结

| 特性 | 实现方式 |
|------|---------|
| 实时多人同步 | Socket.IO Rooms 隔离，每次状态变更广播 `gameStateUpdate` |
